#  QuickRepo — Repository Blueprint Builder
-----------------------------------------------

QuickRepo is an interactive web app built with React that automates writing repetitive Markdown text for newly created code repositories. Fill out your project's basic components and copy-paste a beautiful layout in seconds.

## Live Preview

##  Features

* ⚡ **Instant Generation:** Form updates the text output dynamically as you type.
*  **Array Parsers:** Separating elements with commas automatically creates neat Markdown bullet points.
*  **Clipboard API Sync:** Copies full code with a single click.

##  Tech Stack

* `React`
* `Vite`
* `JavaScript (ES6+)`
